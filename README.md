# Lodgify Technical Interview

## Before the Interview

Please clone this repository and have the project up and running **before** the interview starts.

During the interview, you'll be asked to share your screen and make live improvements to this codebase. There are no tasks to prepare in advance, just make sure the app runs locally and you're comfortable with the code.

## Installation

First, install the project dependencies:

```bash
npm install
```

## Running the Project

To run the development server:

```bash
npm run dev
```

This will start the Vite development server and open the application in your browser.
